# AI PR Review Assistant

AI PR Review Assistant is a tool that automatically analyzes GitHub Pull Requests using Gemini AI and generates code review feedback. It helps developers identify potential bugs, security concerns, performance issues, and code quality improvements before merging changes.

## Features

* Reviews open GitHub Pull Requests
* Analyzes modified files
* Generates AI-powered code review reports
* Highlights bugs, security issues, and improvements
* Supports automated PR review workflows

## How to Use

1. Open the notebook in Google Colab.
2. Run all cells.
3. Enter your GitHub Personal Access Token when prompted.
4. Enter your Gemini API Key.
5. The tool will automatically analyze open Pull Requests and generate review feedback.

## Requirements

* GitHub Personal Access Token
* Gemini API Key
* Google Colab
